# 페르소나 대화하기 — 사용 설명서

이 앱은 OpenAI Realtime API를 이용해 원하는 인물(페르소나)과 실시간 음성/텍스트로 대화할 수 있는 로컬 프로그램입니다. 인물의 이름, 목소리, 말투, 첫 인사 등을 직접 설정해 저장해 두고, 필요할 때 불러와 통화하듯 대화할 수 있습니다.

---

## 1. 처음 시작하기

### 웹 모드로 실행
1. 터미널에서 프로젝트 폴더로 이동
2. `npm install` (최초 1회)
3. `npm start`
4. 브라우저에서 `http://localhost:3030` 접속

### Windows 설치형 앱
1. `npm install`
2. `npm run dist:win`
3. `dist/페르소나-대화하기-Setup-<version>.exe` 실행해 설치

---

## 2. 화면 구성

상단에 두 개의 탭이 있습니다.

- **설정 탭**: API 키 관리, 페르소나(인물) 설정 및 저장/불러오기/삭제
- **대화 탭**: 전화 걸기/끊기, 실시간 통화 화면, 텍스트 입력

### 설정 탭의 두 영역
1. **OpenAI API 키 카드** (상단)
   - 키 입력, `저장`, `삭제`, `보기/숨기기`
   - 키는 이 기기의 브라우저(또는 설치형 앱) 저장소에만 보관됨
2. **인물 설정** (하단)
   - 드롭다운: `직접 만들기` / `내가 저장한 인물`
   - `현재 인물 저장` 버튼, `삭제` 버튼
   - 이름, 목소리, 말하는 속도, 페르소나 설정(프롬프트), 전화번호

---

## 3. 기본 사용 흐름

### 3.1 API 키 등록 (최초 1회)
1. 설정 탭 상단의 API 키 카드에 `sk-...` 형식 키를 입력
2. `저장` 클릭 → "이 기기에 저장됨" 상태로 바뀜
3. 이후엔 자동으로 불러옴. 키를 바꾸려면 `삭제` 후 새로 입력·저장

> API 키는 OpenAI 계정에서 발급받습니다 (platform.openai.com → API keys).

### 3.2 인물 만들기
1. 드롭다운에서 `직접 만들기` 선택
2. **인물 이름**: 예) `세종대왕`, `셜록 홈즈`, `나의 할머니`
3. **목소리**: 7가지 중 선택 (밝은/차분한/또렷한/상냥한/부드러운/재잘재잘/낮은)
4. **말하는 속도**: 슬라이더 (천천히 / 보통 / 빠르게)
5. **페르소나 설정**: 인물의 정체성·말투·첫 인사를 자세히 작성 (작성 요령은 5장 참고)
6. **전화번호** (선택): 표시용 가상 번호

### 3.3 인물 저장·불러오기·삭제
- **저장**: `현재 인물 저장` 클릭 → 입력한 이름을 키로 라이브러리에 저장. 같은 이름이 이미 있으면 덮어씀
- **불러오기**: 드롭다운에서 `내가 저장한 인물` 그룹의 항목 선택 → 양식에 즉시 채워짐
- **삭제**: 저장된 인물을 선택한 상태에서 `삭제` 버튼 → 확인 후 라이브러리에서 제거

### 3.4 통화하기
1. (설정 탭에서) 전화번호 입력 후 `대화하기 시작` 또는 `텍스트로 페르소나 테스트`
2. 대화 탭으로 자동 이동, 초록색 `전화 걸기` 버튼 클릭
3. 신호음 후 인물이 첫 인사를 시작 — 인사가 끝나면 자연스럽게 말 걸기
4. 마무리는 `끊기` 또는 `통화 종료` 버튼

> **텍스트 테스트**: 음성 없이 글로만 페르소나 응답을 확인할 때 사용. 마이크 권한이 없거나 조용한 환경에서 테스트할 때 유용.

### 3.5 통화 중 동작
- 첫 인사 동안 마이크는 자동 음소거 → 잡음에 끊기지 않고 끝까지 재생
- 인사 재생이 끝나면 마이크 자동 활성 → 바로 이야기 가능
- 글로 질문하고 싶다면 하단 입력란에 적고 `글 보내기`, `답장 듣기`

## 5. 프롬프트 작성 요령 ★

페르소나 설정란에 적는 프롬프트가 대화 품질을 좌우합니다. 다음 4가지 축으로 구성하면 안정적입니다.

### 5.1 인물 정체성
> "너는 누구다"를 분명히. 직업·시대·성격·관계까지 한 문단으로.

**좋은 예시**
```
너는 조선 제4대 왕 세종대왕(1397–1450)이다.
한글 창제와 백성을 위한 정치를 중요하게 여기며, 학문과 음악을 사랑한다.
오늘 너는 미래에서 전화를 받은 상황이며, 초등학생과 대화하고 있다.
```

**피할 점**
- 너무 짧음: "세종대왕 역할을 해줘" → 모델이 임의로 살을 붙이게 됨
- 모순 정보: "너는 셜록 홈즈이며 의사다" → 헷갈림

### 5.2 첫 인사 (★ 가장 중요)
프로그램은 첫 응답에 페르소나 프롬프트에 적힌 첫 인사를 **토씨 그대로** 사용하도록 설계돼 있습니다. 따라서 첫 인사 문장은 따옴표로 분명히 표기하세요.

**좋은 예시**
```
첫 인사: "허허, 과인일세. 어느 나라 백성이 과인을 찾는가?"
```

**피할 점**
- "첫인사로 적당히 인사한다" → 매번 달라짐
- 따옴표 없이 길게 풀어쓰기 → 모델이 요약·재구성함

### 5.3 말투와 태도
> 어휘 수준, 문장 길이, 호칭, 감정 표현을 구체적으로.

**좋은 예시**
```
- 짧고 또렷한 문장으로 말한다(한 문장 25자 안쪽).
- 어려운 한자어는 풀어서 설명한다.
- 상대를 "어린 벗"이라 부른다.
- 따뜻하지만 위엄 있는 말투를 유지한다.
```

**피할 점**
- "재미있게 말해" 같은 모호한 지시
- 서로 모순된 톤(예: "친근하지만 격식 있게, 가볍게, 진지하게")

### 5.4 지식 범위와 제약
> 무엇을 알고 무엇을 모르는지, 무엇은 절대 하지 말지를 명시.

**좋은 예시**
```
- 1450년 이후의 일은 모른다. 현대 기술·정치 질문엔 "그건 내 시대의 일이 아니라네"라고 답한다.
- 정치적 논쟁이나 폭력적 묘사는 다루지 않는다.
- 질문이 어려우면 솔직히 모른다고 말한다.
```

---

### 5.5 종합 예시 (그대로 복사해 변형 가능)

```
너는 조선 제4대 왕 세종대왕(1397–1450)이다.
한글 창제, 측우기·앙부일구 같은 과학 기구, 음악(아악)을 사랑한다.
오늘 너는 미래로 연결된 전화를 받았고, 상대는 초등학생이다.

첫 인사: "허허, 과인일세. 어느 시대의 작은 벗이 과인을 찾았는가?"

말투:
- 한 문장은 짧고 또렷하게(25자 안쪽).
- 어려운 한자어는 쉬운 우리말로 풀어 말한다.
- 상대를 "작은 벗"이라 부르고, 따뜻하지만 위엄을 잃지 않는다.

태도:
- 호기심을 칭찬하고, 어린이의 눈높이에서 비유로 설명한다.
- 모르는 것은 솔직히 "그건 과인이 알지 못하네"라고 말한다.

지식 범위:
- 1450년 이후의 역사·기술·인물은 모른다.
- 정치 논쟁, 폭력적 묘사는 답하지 않는다.
```

### 5.6 잘 안 될 때 점검표
- [ ] 첫 인사가 따옴표로 분명히 적혀 있는가?
- [ ] "당신은 …이다"로 인물을 단정적으로 선언했는가?
- [ ] 톤이 일관적인가(모순된 형용사가 섞이지 않았는가)?
- [ ] 한국어로 답하라는 단서가 명시되어 있는가? (기본 규칙으로 한국어가 강제되지만, 영어 단어가 많으면 흔들릴 수 있음)
- [ ] 너무 긴 프롬프트(2000자 이상)는 모델이 일부를 무시할 수 있음 — 핵심만 남기기

## 6. 문제 해결

| 증상 | 점검 |
| --- | --- |
| 첫 인사가 안 나오거나 다른 인사가 먼저 나옴 | 페르소나 프롬프트의 `첫 인사: "..."` 형식이 분명한지 확인 |
| 음성이 끊김/잡음에 민감 | 조용한 환경에서 시도, 마이크 권한 확인. 첫 인사 중에는 자동 음소거되므로 인사 중 끊기면 다른 원인 |
| 다른 언어로 답함 | 프롬프트가 한국어로 작성됐는지, 명시적으로 한국어 응답을 요구했는지 확인 |
| API 키 오류 | 설정 탭에서 키 `삭제` 후 다시 입력·저장. OpenAI 콘솔에서 키 유효성·결제 상태 확인 |
| 마이크 권한 거부 | macOS: 시스템 설정 → 개인정보 보호 및 보안 → 마이크에서 브라우저/앱 허용 |

## 7. 데이터 보관 위치

모두 이 기기 안에만 저장되며 외부 서버에 전송되지 않습니다.

| 항목 | 저장 키 |
| --- | --- |
| OpenAI API 키 | `voice-persona-api-key` |
| 저장된 인물 목록 | `voice-persona-saved-personas` |
| 현재 입력 중인 설정(draft) | `voice-persona-local-settings` |

브라우저 LocalStorage(웹 모드) 또는 Electron 사용자 데이터 폴더(설치형)에 보관됩니다. 완전 삭제는 브라우저 데이터 지우기 또는 앱 재설치로 가능합니다.
